Exploratory Data Analysis (EDA) on Quotes Dataset

Project Description
This project performs Exploratory Data Analysis (EDA) on the `quotes_dataset.csv` dataset using Python. It explores the dataset structure, checks data quality, generates descriptive statistics, visualizes patterns, and identifies potential issues such as missing values, duplicates, and outliers.

Features
- Load the dataset using Pandas.
- Display dataset structure and information.
- Check for missing values.
- Visualize missing values using a heatmap.
- Generate descriptive statistics.
- Detect duplicate records.
- Analyze numerical data distributions.
- Perform correlation analysis.
- Detect outliers using boxplots.
- Perform a sample hypothesis test.
- Generate a data quality report.

 Technologies Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

Installation

1. Clone the repository:

git clone <repository_url>


2. Navigate to the project directory:

cd <project_folder>

3. Install the required libraries:

pip install -r requirements.txt


 Usage

Run the Python script:

python eda.py


 Output
The script provides:
- Dataset overview
- Missing value analysis
- Descriptive statistics
- Duplicate detection
- Distribution plots
- Correlation heatmap
- Outlier detection plots
- Hypothesis testing results
- Data quality report
